Una landing page moderna, minimalista y altamente optimizada para conversión, diseñada específicamente para **Jatech**, un servicio técnico especializado en reparación de celulares y dispositivos móviles ubicado en Iquitos, Loreto.

## 🎯 Sobre el Proyecto

Este proyecto nace con el objetivo de establecer una fuerte presencia digital para Jatech, el emprendimiento liderado por Javier. La página está estratégicamente diseñada para transmitir confianza, profesionalismo y facilitar el contacto rápido con clientes potenciales que necesitan soluciones técnicas para sus dispositivos.

El enfoque principal del diseño es **Mobile-First**, garantizando una experiencia fluida e intuitiva desde cualquier smartphone, con micro-interacciones y animaciones suaves que elevan la percepción de la marca sin sacrificar el rendimiento web.

## ✨ Características Principales

- **Diseño UI/UX Minimalista:** Interfaz limpia en tonos azul corporativo, blanco y oscuro, enfocada en la legibilidad y la acción.
- **Animaciones Nativas (Vanilla):** Implementación de `IntersectionObserver` para efectos de *fade-in* y *slide-up* al hacer scroll, creando una experiencia inmersiva.
- **Optimización de Conversión (CRO):** Llamados a la acción (CTAs) estratégicamente ubicados y un botón flotante de WhatsApp con animación de pulso continuo.
- **Sección de Tracking Simulada:** Interfaz preparada para una futura integración de seguimiento de estado de reparación de equipos mediante número de orden.
- **Totalmente Responsivo:** Adaptación perfecta a cualquier tamaño de pantalla, desde monitores Ultrawide hasta dispositivos móviles pequeños.
- **Arquitectura "Zero-Dependencies":** Construido en un solo archivo `index.html` sin frameworks pesados, garantizando tiempos de carga ultrarrápidos.

## 🛠️ Tecnologías Utilizadas

- **Estructura:** HTML5 Semántico
- **Estilos:** CSS3 Moderno (CSS Variables, Flexbox, CSS Grid, animaciones `@keyframes` personalizadas)
- **Interactividad:** Vanilla JavaScript (DOM Manipulation, Observers, Event Listeners)
- **Tipografía:** Google Fonts ([Inter](https://fonts.google.com/specimen/Inter))
- **Iconografía:** FontAwesome 6
